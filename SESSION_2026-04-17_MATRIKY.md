## Technické poznatky

### URL patterns a stahování

- Matriky.online má JPG na přímém URL (bez autentifikace!)
- **Pozor:** URL pattern se liší podle regionu — **Trutnov** vs. **Jičín** (Žireč)
- Jičín encoding: `Ji%C4%8Din` (č encoded) — NE `Ji%C5%94in`
- Soubory 700 KB – 2 MB, JPG 5000×4000 px

### Transkribus file upload workflow (CDP)

1. Navigovat na `/upload?collId=XXXX`
2. Čekat na `input[type=file]` element
3. `DOM.setFileInputFiles` (Chrome DevTools Protocol)
4. Čekat na text "Upload complete"
5. Klik "Sort" (pokud se zobrazí)
6. Vyplnit text input s titulem dokumentu
7. Klik "Submit" button
8. Čekat na redirect mimo `/upload` URL

### Opravy / problémy během session

- **První Žireč download** — špatný URL pattern (Trutnov vs. Jičín) → stáhnuté HTML error stránky 15 KB místo JPG
- **První Žireč upload** — nedokončil 2 ze 4 dokumentů (title/submit selhal)
- **Transkribus editor** — občasný "Loading editor" stav až několik minut
- **Checkbox multi-select** v dialogu Process with AI — UI stav se nesynchronizuje s `click()` eventy (React controlled inputs)
